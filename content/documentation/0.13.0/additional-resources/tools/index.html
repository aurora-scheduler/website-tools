Tools

Various tools integrate with Aurora. Is there a tool missing? Let us know, or submit a patch to add it!

  • Load-balancing technology used to direct traffic to services running on Aurora:

    • synapse based on HAProxy
    • aurproxy based on nginx
    • jobhopper performs HTTP redirects for easy developer and administrator access

  • RPC libraries that integrate with the Aurora’s service discovery mechanism:

    • linkerd RPC proxy
    • finagle (Scala)
    • scales (Python)

  • Monitoring:

    • collectd-aurora for cluster monitoring using collectd
    • Prometheus Aurora exporter for cluster monitoring using Prometheus
    • Prometheus service discovery integration for discovering and monitoring services running on Aurora

  • Packaging and deployment:

    • aurora-packaging, the source of the official Aurora packages
